# Welcome to your Cabinet Cost Evaluation !!
## Currently at a simple mockup, with tons of room to grow :)
### Available features from Version 1:
  - Customizing the height, width, and depth
  - Picking out your cabinet's material type, finish, and color
  - Getting a 360 preview of your cabinet
  - Seeing live cost prediction updates while you customize

### Features implemented in Version 2: 
  - Customizable shelving
  - Save/export/load
  - IKEA-like room context
  - Option to add your own image for room setup
  - Adds placement mode to place within the chosen 3D space
  - Updated pricing
  - Made aluminium the only option of material
  - Add accordion-style control setup to save space with new additions


### Possible features to be added in Version 3:
  - Drag to control measurements
  - Mobile-friendly
  - Branding
  - Hardware choices (hinges, screws, etc)
  - And more!
